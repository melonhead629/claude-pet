claude-pet

A virtual pet that lives inside Claude Code. Feed it by coding, watch it grow.

Paste into Claude Code to install

Install claude-pet from github.com/amellin794/claude-pet

Or install manually from the repo.

How it works

1

Hooks into Claude Code — session-start and post-tool-use hooks track your activity automatically.

2

Your pet evolves — it gains energy from tool calls, gets hungry over time, and grows through five stages from egg to elder.

3

Check in anytime — a status line shows your pet's mood in every session, or open the dashboard for the full view.
